# Examples

Reference outputs and breakdowns showing what's possible with this toolkit.

## What goes here

- Anonymised before/after pairs (reference image → generated output)
- Winning prompts that consistently produce great results
- Failed prompts and the fix that worked
- Walkthroughs of full campaigns built end-to-end

## Contribute

If you build something with this toolkit and want to share, open a PR with:
- The original brief in plain English
- The full Higgsfield prompt used
- The output URL or asset
- Performance data if you ran it as an ad

Strip any client/customer identifying info before submitting.
